## 작업 절차

### 1단계: 데이터 로드

src/data/questions.ts를 읽고 모든 문제 데이터를 파악합니다.

### 2단계: 통계 산출 및 출력

아래 항목을 순서대로 출력합니다.

#### 2-1. 전체 요약

```
## 전체 요약

| 항목 | 값 |
|------|-----|
| 총 문제 수 | |
| 카테고리 수 | |
| 카테고리당 문제 수 | (균등 여부 표시) |
| 최대 가능 점수 | (scoring.ts의 MAX_SCORE_PER_QUESTION × 총 문제 수) |
```

#### 2-2. 카테고리별 문제 수

```
## 카테고리별 문제 수

| 카테고리 | 문제 수 | ID 범위 | 비율 |

```

#### 2-3. 정답 분포

카테고리별 및 전체 정답 번호(0~3) 분포를 출력합니다.

```
## 정답 분포

### 전체

| 정답 번호 | 개수 | 비율 | 막대 |
|----------|------|------|------|
| 0 | | | ████░░░░ |
| 1 | | | ████████ |
| 2 | | | ██████░░ |
| 3 | | | ██░░░░░░ |

이상적 분포: 각 25%

### 카테고리별

| 카테고리 | 0 | 1 | 2 | 3 | 편향도 |
|---------|---|---|---|---|-------|
| 한국사 | | | | | 균등 / 편향(N번 과다) |
| 과학 | | | | | |
| 지리 | | | | | |
| 일반상식 | | | | | |
```

- 편향도: 특정 번호가 전체의 40% 이상이면 "편향" 표시

#### 2-6. 개선 제안
