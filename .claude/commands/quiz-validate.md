src/data/questions.ts 파일에서 퀴즈 문제(question, explanation 필드)를 분석하여 최상급 표현을 찾아 보고해 주세요.

## 검증 범위

- `$ARGUMENTS`가 지정된 경우: 해당 카테고리(한국사, 과학, 지리, 일반상식)의 문제만 검증
- `$ARGUMENTS`가 비어 있는 경우: 전체 문제를 검증

## 검색 대상 키워드

- 가장
- 최초
- 최대
- 최고
- 최다
- 최소
- 최저
- 최장
- 최단
- 세계 제일
- 유일

## 출력 형식

각 문제에 대해 다음 형식으로 출력해 주세요:

```
### 최상급 표현이 포함된 문제 목록

| # | ID | 카테고리 | 최상급 표현 | 문제 내용 (요약) | 기준 명시 여부 |
|---|-----|---------|-----------|----------------|-------------|
```

## 검증 기준

CLAUDE.md의 "퀴즈 문제 교차 검증 가이드라인"에 따라 각 문제를 검증합니다:

1. **기준 명시 여부**: "가장 큰", "최초의" 등 표현에 측정 기준이 명시되어 있는지 확인
2. **시점 명시 여부**: 시간에 따라 변할 수 있는 정보에 시점이 명시되어 있는지 확인
3. **범위 한정 여부**: 지리적·분류적 범위가 명확히 한정되어 있는지 확인

마지막에 개선이 필요한 문제가 있으면 구체적인 개선 제안을 추가해 주세요.
