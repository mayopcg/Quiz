## 분석 대상 파일

- `src/hooks/useRanking.ts` — 순위 데이터 관리 훅
- `src/components/RankingScreen.tsx` — 순위 화면 UI
- `src/components/ResultScreen.tsx` — 결과 화면 (순위 연동, getGrade 함수)
- `src/hooks/useGameState.ts` — 게임 상태 관리 (PlayerResult 생성)
- `src/types/index.ts` — PlayerResult, CategoryScore 타입
- `src/App.tsx` — 컴포넌트 간 데이터 전달 구조
