관리자 통합 보고서를 생성합니다. 채점 체계, 문제 난이도, 순위 시스템, 성적 시뮬레이션을 모두 분석하여 종합 보고서를 출력합니다.

**이 명령은 4개의 개별 분석을 순서대로 수행한 뒤, 종합 결론을 도출합니다.**

## 분석 대상 파일

- `src/utils/scoring.ts` — 점수 계산 함수 및 상수
- `src/components/ResultScreen.tsx` — 등급 판정 로직
- `src/hooks/useRanking.ts` — 순위 데이터 관리
- `src/hooks/useGameState.ts` — 게임 상태 및 점수 산출
- `src/components/RankingScreen.tsx` — 순위 화면 UI
- `src/data/questions.ts` — 전체 퀴즈 문제 데이터
- `src/types/index.ts` — 타입 정의
- `src/App.tsx` — 컴포넌트 간 데이터 전달
