src/data/questions.ts 파일에서 ID $1번부터 $2번까지의 퀴즈 문제를 검토해 주세요.

## 검토 항목

### 1. 난이도 분석

각 문제의 난이도를 다음 기준으로 평가해 주세요:

- **상**: 전문 지식이 필요하거나, 선택지 간 변별이 어려운 문제
- **중**: 일반 교양 수준에서 풀 수 있지만 약간의 사고가 필요한 문제
- **하**: 대부분의 사람이 쉽게 풀 수 있는 기본 상식 문제

#### 출력 형식

```
| # | ID | 카테고리 | 문제 내용 (요약) | 난이도 | 난이도 판단 근거 |
|---|-----|---------|----------------|-------|----------------|
```

난이도 분석 후 전체 분포를 요약해 주세요:
- 상/중/하 각 몇 문제인지
- 난이도 분포가 적절한지 의견 제시

### 2. 정답 분포 분석

해당 범위의 문제들에서 정답 번호(answer 필드, 0~3)의 분포를 확인해 주세요.

#### 출력 형식

```
| 정답 번호 | 보기 위치 | 해당 문제 수 | 비율 | 해당 문제 ID |
|----------|----------|------------|------|------------|
| 0        | 첫 번째   |            |      |            |
| 1        | 두 번째   |            |      |            |
| 2        | 세 번째   |            |      |            |
| 3        | 네 번째   |            |      |            |
```

정답 분포 분석 후:
- 특정 번호에 정답이 과도하게 집중되어 있는지 확인
- 편향이 있을 경우 정답 위치를 변경할 문제를 구체적으로 제안

### 3. 종합 의견

- 난이도 밸런스에 대한 종합 평가
- 정답 분포 균등성에 대한 종합 평가
- 개선이 필요한 문제가 있으면 구체적인 개선 제안
