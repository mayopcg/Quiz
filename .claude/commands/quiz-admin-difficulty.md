각 문제의 예상 난이도를 분석하고, 카테고리별 난이도 균형을 점검해 주세요.

## 분석 대상 파일

- `src/data/questions.ts` — 전체 퀴즈 문제 데이터

## 작업 절차

### 1단계: 난이도 판정 기준

다음 요소를 종합하여 각 문제의 난이도를 상/중/하로 판정합니다:

- **문제 길이**: 긴 문제는 읽는 시간이 많이 필요 → 난이도 상승 요인
- **보기 유사도**: 보기 간 구분이 어려우면 → 난이도 상승 요인
- **전문 용어**: 전문적/학술적 용어 포함 → 난이도 상승 요인
- **일상 접근성**: 일상에서 쉽게 접할 수 있는 지식 → 난이도 하락 요인
- **오답 매력도**: 오답이 그럴듯하면 → 난이도 상승 요인
